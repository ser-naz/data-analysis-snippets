Data-Snippets

A collection of Python scripts demonstrating fundamentals of data manipulation, cleaning, and visualization. 
This repository serves as a reference and practice ground for common data tasks and techniques used in real-world projects.

This project is designed to:
- Provide reusable code snippets for working with data using Python.
- Demonstrate basic data handling techniques with **NumPy, Pandas, and Matplotlib**.
- Act as a cheat sheet for future projects or job-related tasks.

The repository includes examples on:
- NumPy: arrays, operations, reshaping.
- Pandas: data cleaning, filtering, merging, and date handling.
- Visualization: bar charts, scatter plots, pie charts, heatmaps.
